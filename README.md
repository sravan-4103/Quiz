# 🌍 Global Map Quiz App

A modern, responsive, and interactive Quiz Application designed to test geographic knowledge. This project features a sleek dark-themed UI and dynamic question handling using Vanilla JavaScript.

---

## ✨ Key Features

* **Dynamic UI:** Smooth transitions between Start, Quiz, and Result screens using a single-page approach.
* **Progress Tracking:** An animated progress bar that visually indicates how far you are in the quiz.
* **Real-time Feedback:** * Correct answers highlight in **green**.
    * Incorrect choices highlight in **red** while showing the correct answer simultaneously.
* **Smart Scoring:** Tracks your score throughout the session and provides a personalized summary at the end.
* **Mobile First:** Fully responsive design that adapts to mobile, tablet, and desktop screens.
* **Dark Mode Aesthetic:** Built with a high-contrast dark theme for better readability and eye comfort.

---

## 🛠️ Tech stack

* **HTML5:** Structure and semantics.
* **CSS3:** Flexbox layout, custom animations, and media queries for responsiveness.
* **JavaScript (ES6+):** DOM manipulation, event handling, and state management.
